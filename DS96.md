# Написание запросов с различными типами соединений
### Будем использовать следующую базу otus_ddl
```sql
CREATE DATABASE otus_dql;
```
### Создадим таблицы
```sql
CREATE TABLE buses (
    bus_id serial,
    name   varchar
);
CREATE TABLE drivers (
    driver_id   serial,
    first_name  varchar,
    second_name varchar
);


CREATE TABLE routes (
    route_id  serial,
    name      varchar,
    bus_id    int,
    driver_id int
);
```
### Заполним таблицы данными
